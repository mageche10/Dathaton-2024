Utilització:

  - core.py serveix per crear models senzills, que únicament analitzen imatges
  - very_updated_core.py crea models que també te en compte algunes dades de "product_data"
  - en ambdós scripts s'ha de tindre cura amb els inputs
  - la interfície gràfica s'inicialitza amb "streamlit run interface.py"
    - per que l'API funcioni s'ha d'activar amb "python api.py" i potser activar el botó de demo
  - és possible que s'hagi d'instalar algunes dependències que no hi són pròpies del Python
  - tot fitxer relacionat amb utilitzar product_data.csv està marcat amb very_updated_
